About me

I am a dedicated AEM architect and practitioner. The technical lead for the Adobe Document Cloud AEM web presence featuring a cornerstone technology of today's business process: Adobe Acrobat.

To be an Adobe Experience Manager developer is to be a full stack engineer. I am am Oracle certified Java Engineer with a deep understanding of the AEM APIs and the underlying frameworks that compose AEM.

Milliseconds matter when you're marketing through the Internet. A strong web developer background allows me to focus on providing optimized and uninhibited user experiences. I create fast, mobile responsive, SEO friendly websites using HTML5, CSS3, JavaScript, and the latest tools in client-side development.

As an experienced software development professional, I have acquired a profound respect for the Agile process. I hold certifications for both ScrumMaster and Product Owner from the Agile Alliance.
